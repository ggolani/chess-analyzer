# Chess Analyzer - Design Goals

This project aims to provide a high-level analysis of a user's recent chess games.

## Core Objectives
- **Fetch Recent Games:** Automatically retrieve the last 10 games played by a specific user from Chess.com.
- **Grandmaster-Level Annotation:** Use LLM prebuilt knowledge to provide per-move annotations at a Grandmaster level.
- **Move Strength Analysis:** Integrate Stockfish to evaluate and display the strength/evaluation of each move.
- **Interactive UI:** Present the annotated games and evaluations in a user-friendly interface.
